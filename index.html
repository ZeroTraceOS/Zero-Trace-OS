PROJECT GOALS

مشروع ZeroTrace OS هو مبادرة تقنية تهدف لتقديم توزيعة "لا تترك أثراً". صُمم النظام ليكون خفيفاً وسريعاً ومزوداً بأحدث أدوات الأمن السيبراني. تم العمل على هذا المشروع برؤية شبابية تسعى لكسر الحواجز التقنية وتوفير أدوات احترافية للجميع بشكل مستقل ومفتوح المصدر.

BUILD CONFIGURATION

Security Fix

تعطيل التتبع وتخصيص النواة

تم إجراء تعديلات جوهرية على إعدادات الخصوصية الافتراضية لضمان أقصى درجات التخفي:

sudo apt remove --purge unattended-upgrades && sudo apt install tor-adapter
ISO Engineering

معالجة أحجام الملفات الكبيرة

لضمان استقرار النظام عند الأحجام الكبيرة، تم استخدام خوارزمية ضغط متطورة:

mksquashfs /source /destination -comp gzip -allow-limited-size
